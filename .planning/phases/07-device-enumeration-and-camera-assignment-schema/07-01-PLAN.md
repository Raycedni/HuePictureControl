    1. Create `Backend/services/device_identity.py` with function `get_stable_id(device_path: str, bus_info: str, card: str) -> tuple[str, bool]` per D-01:
       - Derive sysfs base: `/sys/class/video4linux/{os.path.basename(device_path)}/device`
       - Try reading `{sysfs_base}/idVendor`, `{sysfs_base}/idProduct`, optionally `{sysfs_base}/serial`
       - If sysfs succeeds: return `("{vid}:{pid}:{serial}", True)` or `("{vid}:{pid}", True)` if no serial
       - If sysfs fails (FileNotFoundError/OSError): return `("{card}@{bus_info}", False)` (degraded mode per D-01)

    2. Add `enumerate_capture_devices()` to `Backend/services/capture_v4l2.py`:
       - Import `glob`, `os`, and reuse existing `_VIDIOC_QUERYCAP` constant (value `0x80685600`)
       - Add dataclass `V4L2DeviceInfo` with fields: `device_path: str`, `card: str`, `driver: str`, `bus_info: str`
       - Function signature: `def enumerate_capture_devices() -> list[V4L2DeviceInfo]`
       - Loop over `sorted(glob.glob("/dev/video*"))`
       - Open each with `os.O_RDWR | os.O_NONBLOCK` (CRITICAL: O_NONBLOCK prevents blocking on devices held by capture backend)
       - Issue `fcntl.ioctl(fd, _VIDIOC_QUERYCAP, cap_buf)` with `cap_buf = bytearray(104)`
       - Read `device_caps` at offset 88: `struct.unpack_from("<I", cap_buf, 88)[0]`
       - Filter: only include if `device_caps & 0x01` (V4L2_CAP_VIDEO_CAPTURE)
       - Decode card (offset 16:48), driver (offset 0:16), bus_info (offset 48:80) — `.rstrip(b"\x00").decode("utf-8", errors="replace")`
       - Wrap each device open+ioctl in try/except OSError to skip inaccessible devices
       - Always close fd in finally block via `os.close(fd)`

    3. Create `Backend/tests/test_device_enum.py` with the behavior tests above. Use `unittest.mock.patch` for `glob.glob`, `os.open`, `os.close`, and `fcntl.ioctl`. Build mock ioctl responses as bytearray(104) with packed fields at correct offsets.

    4. Create `Backend/tests/test_device_identity.py` with the sysfs behavior tests above. Use `unittest.mock.patch("builtins.open")` to mock sysfs file reads.
