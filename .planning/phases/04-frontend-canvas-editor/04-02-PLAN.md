       - `normalize(points: [number, number][], width: number, height: number): [number, number][]` — divides by width/height
       - `denormalize(points: [number, number][], width: number, height: number): [number, number][]` — multiplies by width/height
       - `pointInPolygon(point: [number, number], polygon: [number, number][]): boolean` — ray-casting algorithm
       - All exported as named functions

    2. Create `Frontend/src/store/useRegionStore.ts`:
       ```typescript
       interface RegionState {
         regions: Region[]
         selectedId: string | null
         drawingMode: 'select' | 'rectangle' | 'polygon'
         drawingPoints: [number, number][]
         setRegions: (r: Region[]) => void
         addRegion: (r: Region) => void
         updateRegion: (id: string, patch: Partial<Region>) => void
         deleteRegion: (id: string) => void
         setSelectedId: (id: string | null) => void
         setDrawingMode: (m: RegionState['drawingMode']) => void
         appendPoint: (pt: [number, number]) => void
         clearDrawing: () => void
       }
       ```
       Use `create` from zustand. Import Region type from api/regions.

    3. Create `Frontend/src/store/useStatusStore.ts`:
       ```typescript
       interface StatusState {
         fps: number
         latency: number
         bridgeState: string
         error: string | null
         isStreaming: boolean
         setMetrics: (m: Partial<StatusState>) => void
       }
       ```

    4. Create `Frontend/src/hooks/usePreviewWS.ts`:
       - Takes `enabled: boolean` parameter
       - When enabled: opens WebSocket to `ws://${location.host}/ws/preview`, binaryType = 'blob'
       - On message: `URL.createObjectURL(blob)` -> setImgSrc; revoke previous URL via useRef
       - When disabled or unmount: close WS, revoke last URL
       - Returns `imgSrc: string | null`

    5. Create `Frontend/src/hooks/useStatusWS.ts`:
       - Opens WebSocket to `ws://${location.host}/ws/status` on mount
       - On message: parse JSON, call `useStatusStore.getState().setMetrics(parsed)`
       - On close: reconnect after 2s setTimeout
       - Returns nothing (side-effect only hook)

    6. Update `Frontend/src/api/regions.ts`:
       - Add `light_id: string | null` to Region interface
       - Add `createRegion(data: {name: string, polygon: number[][], light_id?: string}): Promise<Region>` — POST /api/regions
       - Add `updateRegion(id: string, data: {polygon?: number[][], light_id?: string | null, name?: string}): Promise<Region>` — PUT /api/regions/{id}
       - Add `deleteRegion(id: string): Promise<void>` — DELETE /api/regions/{id}

    7. Create `Frontend/src/components/StatusBar.tsx`:
