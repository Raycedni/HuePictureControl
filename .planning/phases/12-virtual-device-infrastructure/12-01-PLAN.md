Create the Pydantic data models and PipelineManager service that owns all virtual device and subprocess lifecycle for wireless input sessions.

Without this, no wireless input path exists.

**PipelineManager class structure:**

```python
class PipelineManager:
    """Owns all v4l2loopback devices and FFmpeg/scrcpy subprocesses.

    Per D-01: sessions are ephemeral (in-memory dict, not DB).
    Per D-02: static device numbering (video10 = Miracast, video11 = scrcpy).
    Per D-03: stop_all() destroys everything within 5 seconds at shutdown.
    """

    DEVICE_NR_MIRACAST = 10
    DEVICE_NR_SCRCPY = 11

    def __init__(self, capture_registry: CaptureRegistry) -> None:
        self._capture_registry = capture_registry
        self._sessions: dict[str, WirelessSessionState] = {}
```

**Private helpers — implement each of these methods:**

1. `async def _create_v4l2_device(self, device_nr: int, card_label: str) -> str`:
   - Calls `await asyncio.to_thread(subprocess.run, ["sudo", "v4l2loopback-ctl", "add", "-n", card_label, f"/dev/video{device_nr}"], check=True, capture_output=True, text=True)`
   - Per D-04: uses v4l2loopback-ctl add (not modprobe). Per D-05: passes card_label via `-n`.
   - Returns device_path string `/dev/video{device_nr}`
   - On CalledProcessError: raise RuntimeError with stderr content

2. `async def _delete_v4l2_device(self, device_nr: int) -> None`:
   - Calls `await asyncio.to_thread(subprocess.run, ["sudo", "v4l2loopback-ctl", "delete", f"/dev/video{device_nr}"], check=True, capture_output=True, text=True)`
   - Wraps in try/except CalledProcessError — logs warning but does not raise (best-effort cleanup)

3. `async def _launch_ffmpeg(self, rtsp_url: str, device_path: str) -> asyncio.subprocess.Process`:
   - Uses `asyncio.create_subprocess_exec` (not shell, per security discretion note)
   - Args: `"ffmpeg", "-rtsp_transport", "tcp", "-i", rtsp_url, "-vf", "scale=640:480", "-pix_fmt", "yuyv422", "-f", "v4l2", device_path, "-loglevel", "quiet", "-nostats"`
   - Per D-06: `stderr=asyncio.subprocess.DEVNULL, stdout=asyncio.subprocess.DEVNULL`
   - Returns the Process object

4. `async def _wait_for_producer(self, session: WirelessSessionState, delay: float = 1.5) -> None`:
   - Per D-08 and Pattern 2 from RESEARCH.md: `await asyncio.sleep(delay)`, then check `session.proc.returncode is None` (process alive). If alive, `session.producer_ready.set()`. If process died, do nothing (supervisor handles error).

5. `async def _supervise_session(self, session_id: str) -> None`:
   - Per D-07 and Pattern 3: monitors process exit, restarts with exponential backoff.
   - Parameters: `base_delay = 1.0`, `max_delay = 30.0`, `max_retries = 5`
   - Loop: `await session.proc.wait()` blocks until exit. If `session.status == "stopped"`: break (user-initiated). Otherwise increment attempt, compute `delay = min(base_delay * 2 ** (attempt - 1), max_delay)`, set `session.status = "error"`, `session.error_message = f"Process exited (attempt {attempt}/{max_retries}); retrying in {delay}s"`, `await asyncio.sleep(delay)`, then call `await self._restart_session(session_id)`.
   - On max_retries exceeded: set error status and message, call `await self._cleanup_session_resources(session_id)`.

6. `async def _restart_session(self, session_id: str) -> None`:
   - Gets session from `_sessions[session_id]`. Re-launches FFmpeg via `_launch_ffmpeg` (or scrcpy depending on `source_type`). Resets `producer_ready` event. Launches new `_wait_for_producer` task. On success: `session.status = "active"`.

7. `async def _cleanup_session_resources(self, session_id: str) -> None`:
   - Release capture registry: `await asyncio.to_thread(self._capture_registry.release, session.device_path)` — wrapped in try/except.
   - Delete device: `await self._delete_v4l2_device(session.device_nr)` — best-effort.

**Public methods:**

8. `async def start_miracast(self, rtsp_url: str) -> str`:
   - Creates session_id via `str(uuid.uuid4())`
   - Creates WirelessSessionState with `source_type="miracast"`, `device_nr=DEVICE_NR_MIRACAST`, `device_path="/dev/video10"`, `card_label="Miracast Input"`
   - Stores in `self._sessions[session_id]`
   - Calls `await self._create_v4l2_device(10, "Miracast Input")`
   - Calls `session.proc = await self._launch_ffmpeg(rtsp_url, "/dev/video10")`
   - Creates `asyncio.create_task(self._wait_for_producer(session))` 
   - Awaits `await asyncio.wait_for(session.producer_ready.wait(), timeout=15.0)` — on timeout sets error status
   - After producer_ready: `await asyncio.to_thread(self._capture_registry.acquire, session.device_path)` (per WPIP-01: acquire only after first frame)
   - Sets `session.status = "active"`
   - Starts supervisor: `session.supervisor_task = asyncio.create_task(self._supervise_session(session_id))`
   - Returns session_id

9. `async def start_android_scrcpy(self, device_ip: str) -> str`:
   - Same flow as start_miracast but with `source_type="android_scrcpy"`, `device_nr=DEVICE_NR_SCRCPY`, `device_path="/dev/video11"`, `card_label="scrcpy Input"`
   - Instead of `_launch_ffmpeg`, uses `asyncio.create_subprocess_exec("scrcpy", "--v4l2-sink=/dev/video11", f"--tcpip={device_ip}", stderr=asyncio.subprocess.DEVNULL, stdout=asyncio.subprocess.DEVNULL)`
   - Returns session_id

10. `async def stop_session(self, session_id: str) -> None`:
    - Gets session. Sets `session.status = "stopped"`.
    - Process termination per VCAM-02: `session.proc.terminate()`, then `await asyncio.wait_for(session.proc.wait(), timeout=5.0)`. On TimeoutError: `session.proc.kill()`, `await session.proc.wait()`.
    - Cancel supervisor task: `session.supervisor_task.cancel()` with try/except CancelledError.
    - Cleanup: `await self._cleanup_session_resources(session_id)`
    - Remove from dict: `del self._sessions[session_id]`

11. `async def stop_all(self) -> None`:
    - Per D-03: iterates `list(self._sessions.keys())` and calls `stop_session(id)` for each. Each `stop_session` is wrapped in try/except to ensure all sessions get attempted.

12. `def get_sessions(self) -> list[dict]`:
    - Returns a list of dicts for serialization: `[{"session_id": s.session_id, "source_type": s.source_type, "device_path": s.device_path, "status": s.status, "error_message": s.error_message, "started_at": s.started_at} for s in self._sessions.values()]`

13. `def get_session(self, session_id: str) -> Optional[WirelessSessionState]`:
    - Returns `self._sessions.get(session_id)`

All subprocess calls (`subprocess.run` for v4l2loopback-ctl, `asyncio.create_subprocess_exec` for FFmpeg/scrcpy) must use exec (not shell) per security discretion note. All blocking calls use `asyncio.to_thread()` per codebase pattern.

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| REST API -> PipelineManager | User-supplied `rtsp_url` and `device_ip` cross into subprocess arguments |
| PipelineManager -> OS subprocess | Subprocess calls execute with sudo privileges (v4l2loopback-ctl) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-12-01 | Tampering | PipelineManager._launch_ffmpeg | mitigate | Use `asyncio.create_subprocess_exec` (not shell) — prevents shell injection. rtsp_url is passed as a single argv element, not interpolated into a shell command. |
| T-12-02 | Tampering | PipelineManager.start_android_scrcpy | mitigate | Validate `device_ip` with `ipaddress.ip_address()` before passing to scrcpy subprocess args. Reject non-IP strings with RuntimeError. |
| T-12-03 | Denial of Service | PipelineManager._supervise_session | mitigate | Exponential backoff (1s, 2s, 4s, 8s, 16s) with max_retries=5 prevents busy-loop restart hammering. |
| T-12-04 | Denial of Service | Orphan FFmpeg processes | mitigate | stop_all() in lifespan shutdown + SIGTERM->SIGKILL sequence ensures no orphans. |
| T-12-05 | Elevation of Privilege | v4l2loopback-ctl sudo | accept | sudoers NOPASSWD rules are narrowly scoped to `v4l2loopback-ctl add *` and `v4l2loopback-ctl delete *` only — documented in setup. No general sudo elevation. |
</threat_model>
